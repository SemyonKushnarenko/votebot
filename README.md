# football-bot

Telegram бот на Node.js для планирования сообщений в группы с интерактивными кнопками (50/50, Буду, Не буду), хранением данных в SQLite и авто-редактированием сообщения (список участников + средняя цена).

## Установка

1) Установите Node.js LTS.
2) В корне проекта:

```bash
npm i
```

3) Создайте `.env` по примеру `.env.example`:

- `BOT_TOKEN`: токен Telegram бота
- `SUPERADMINS` (опционально): список `user_id` через запятую
- `TZ` (опционально): таймзона для ввода даты/времени (по умолчанию `Europe/Moscow`)

## Запуск

```bash
npm start
```

## Как пользоваться

- Добавьте бота в группу и дайте права на отправку сообщений и редактирование сообщений.
- В личке с ботом или в любой чат отправьте `/new` (доступ только админам группы).
- Бот пошагово попросит: группу → дату (DD-MM) → время (HH:MM) → цену → текст → предпросмотр → подтверждение.
- В указанное время бот отправит сообщение в группу и будет редактировать его после нажатий кнопок.
